# Agenda-JS

Trabalho de Programação Orientada a Objetos
Implementação em Node.js usando classes, lista de contatos

## Descrição
Este projeto implementa uma Agenda Telefônica utilizando JavaScript (Node.js).
O sistema permite:

- Inserir contatos

- Alterar automaticamente caso o nome já exista

- Buscar contatos pelo nome

- Remover contatos
  
- Listar todos os contatos

- Salvar a agenda em arquivo

- Carregar a agenda de arquivo

## Como Rodar

Instale o Node.js

Baixe o arquivo do projeto

No terminal, dentro da pasta do projeto, execute:
```bash
node agenda.js
```
